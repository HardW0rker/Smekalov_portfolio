Уроки Итоговый тест Справка
Программа представляет собой симулятор CPDLC, в котором пользователю предстоит выполнять задания, рассчитанные на получение первичного представления об использовании типичных процедур CPDLC, а так же, проходить итоговый тест, основанный на предыдущих заданиях для закрепления полученной информации.
